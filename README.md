# 👻 Mitos y Leyendas de Venezuela - Proyecto Escolar

Este sitio web es un proyecto colaborativo para la materia de **Programación**, donde exploramos el folklore venezolano utilizando herramientas de desarrollo profesional.

## 📁 Estructura del Proyecto
El sitio consta de 5 secciones principales:
1. **Inicio (`index.html`):** Menú principal de navegación.
2. **La Sayona (`sayona.html`):** Historia y características.
3. **El Silbón (`silbon.html`):** Relato y curiosidades.
4. **La Loca Luz Caraballo (`loca-luz.html`):** Leyenda andina.
5. **Tabla Comparativa (`tabla.html`):** Resumen de espantos y regiones.

## 🛠️ Tecnologías Usadas
* **Lenguaje:** HTML5 puro.
* **Control de Versiones:** GitHub (Desarrollo colaborativo).
* **Despliegue:** GitHub Pages para la visualización en vivo.
